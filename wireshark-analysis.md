# 📡 Wireshark Traffic Analysis

## 🎯 Objective
To analyze network traffic and identify insecure data transmission.

---

## 🛠️ Tool Used
Wireshark  

---

## 📊 Captured Protocols
- HTTP  
- DNS  
- FTP  

---

## 🔍 Observations

### HTTP Traffic
- Data transmitted in plain text  

---

### DNS Traffic
- Domain names resolved to IP addresses  

---

### FTP Traffic (Important)
- Username and password were visible in plain text  

---

## ⚠️ Security Insight

FTP is insecure because it transmits credentials without encryption.

---

## 📌 Conclusion

Wireshark analysis demonstrated how unencrypted protocols expose sensitive information, highlighting the importance of secure communication methods.
