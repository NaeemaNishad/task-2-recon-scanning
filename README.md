# 🔍 Reconnaissance, Scanning & Traffic Analysis Lab

## 📌 Objective
To learn and perform reconnaissance, port scanning, vulnerability assessment, and network traffic analysis using industry tools.

---

## 🛠️ Tools Used
- Kali Linux  
- Metasploitable2  
- Nmap  
- OpenVAS  
- Wireshark  
- Netcat  

---

## 🔎 Reconnaissance

### Passive Recon
- Whois  
- Nslookup  
- Google Dorking  
- Shodan  

### Active Recon
- Ping Sweep using Nmap  
- Banner Grabbing using Netcat  

---

## 🌐 Port & Service Scanning

Performed using Nmap:
- Basic Scan  
- TCP SYN Scan  
- UDP Scan  
- Service Version Detection  
- OS Detection  

---

## 🛡️ Vulnerability Scanning

Performed using OpenVAS:
- Target: Metasploitable2  
- Scan Type: Full and Fast  
- Identified vulnerabilities categorized as:
  - Critical  
  - High  
  - Medium  
  - Low  

---

## 📡 Packet Analysis

Performed using Wireshark:
- Captured HTTP, DNS, and FTP traffic  
- Observed plaintext credentials in FTP  

---

## 🔥 Firewall Basics

Used iptables to:
- Allow specific ports  
- Block specific ports  
- View active rules
